[DermaGlow](https://dl.acm.org/doi/abs/10.1145/3729474}) demonstrated that skin tone affects SpO$_{2}$ readings. This project investigates the extent to which skin tone influences SpO$_{2}$. Specifically, how does skin tone affect blood oxygen saturation? Which skin tone components contribute to these differences, and to what degree? Can you develop an algorithm to counteract these differences? You will use a test spectroscopy device, the [Lumos](https://dl.acm.org/doi/10.1145/3569502) prototype, in this project.

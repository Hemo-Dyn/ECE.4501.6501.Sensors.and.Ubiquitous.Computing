This project investigates how to increase the operating speed of the AS7341 photodetector chip developed by ams. The photodetector includes 10 channels; however, the chip currently reads all channels together rather than operating on individual channels. This limits the sampling rate from a potential 50 Hz to approximately 14 Hz. You will be provided with the chip documentation and the open-source repository for its library. Your task is to analyze the existing library and develop an optimized usage library that enables faster operation on single channels. You will be provided with devices, Arduino code to operate them, and an API to receive data through a serial channel. For more information, please contact the TA or Professor Amanda Watson. You will use a test spectroscopy device, the [Lumos](https://dl.acm.org/doi/10.1145/3569502) prototype, in this project.

# Sensors and Ubiquitous Computing (ECE 4501/6501)

> Course repository for experiments, projects, and documentation related to sensor systems, embedded platforms, and ubiquitous computing.

![License](https://img.shields.io/badge/license-MIT-blue)

## Projects

The following project folders are included under `Projects/`:

- **00-Glass** – project involving sensor interactions with glass materials  
- **01-IntensityCalibration** – calibration routines for sensor intensity measurements  
- **02-SpO2andSkinTone** – SpO2 measurement and skin tone calibration considerations  
- **03-SmartWatchComparison** – comparison of smartwatch sensor outputs  
- **04-FastAS7341** – Fast sampling with the AS7341 spectral sensor  
- **05-NANDFlashMemory** – NAND flash memory interfacing and tests  
- **06-RobotCar** – robot car platform project

Each folder should contain its own README describing how to build, run, and test that project.

## Getting Started

To get a local copy of the repository:

```
git clone https://github.com/WatsonResearchLabOrganization/ECE.4501.6501.Sensors.and.Ubiquitous.Computing.git
cd ECE.4501.6501.Sensors.and.Ubiquitous.Computing
```

Review the individual project folders for specific setup and usage instructions.

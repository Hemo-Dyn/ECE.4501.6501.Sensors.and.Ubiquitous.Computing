This project investigates the effect of glass on spectroscopy devices. Because infrared LED emitters (e.g., 1650 nm) must comply with photobiological safety exposure limits and risk classifications, devices using infrared channels typically incorporate a transparent barrier (e.g., a glass layer) to limit direct skin exposure and reduce thermal and optical hazard risk. However, we are interested in examining the effect of these glass layers on the readings, spanning both visible light and infrared channels. You will be provided with devices, Arduino code to operate them, and an API to receive data through a serial channel.
